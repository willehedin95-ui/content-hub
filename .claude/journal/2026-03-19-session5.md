- **Competitor swipe image count trim** — `swipe-competitor.ts` generated images for ALL `parsed.image_prompts` without limit. Claude sometimes returns more than requested. Fix: trim to `competitorImageUrls.length * 3`.
- **Backlog cleanup** — Removed P1.5 items (test page testing, live-test competitor swipe) per user request. Moved safe zone overlay from P1 to done.

## Decisions made
- **Safe zone percentages**: Top 14%, bottom 20% based on Stories/Reels UI measurements. These are approximate — actual coverage varies by device.
- **All tab + 9:16**: Show any completed 9:16 translation (picks first available language) rather than the 4:5 original. There's no 9:16 "original" since source images are always 4:5.
- **"9:16 pending" label**: Shows on ALL tabs (removed `activeTab !== "all"` guard) so users always know when outpainting hasn't completed.
- **Text safe zone is prompt-only**: The instruction tells Kie AI where to place text. E2E test confirmed it works, but it's not enforced programmatically — bad placement can still happen occasionally.
