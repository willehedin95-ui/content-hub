## Decisions made
- **Modify existing endpoint vs new endpoint**: Modified the existing `/api/ad-spy/swipe-video` to make `gethookd_ad_id` optional rather than creating a separate `/api/ad-spy/swipe-video-manual`. Simpler, same pipeline, fewer routes.
- **Skip discovered_ads for manual uploads**: Manual uploads don't have a GetHookd ad ID, so they bypass the `discovered_ads` tracking entirely. They still create a `video_job` with `source: "manual"`.
- **100MB upload limit**: Videos are much larger than images (20MB limit for temp uploads), so set a generous 100MB limit.
