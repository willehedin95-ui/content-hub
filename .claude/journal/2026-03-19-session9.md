# Session: 2026-03-19 (session 9)

## What was done
- Implemented **cold start mode** for the Launch Pad auto-push system
- When a market has 0 active ad sets, system now pushes 5 concepts instead of 3 (to find winners faster)
- Added **7-day cooldown** after cold start push — cron skips the market until ads have 7+ days of performance data
- Added 3 visual states to the Launch Pad summary card:
  - **Blue card** = cold start (no active ads, will push 5)
  - **Purple card** = cooldown (recently pushed, X days left)
  - **Green/amber/gray** = normal compression-based mode (existing)
- Budget recommendation: when cold start budget is >1.5x what's needed, shows amber tip (e.g. "2500 SEK but only ~1000 needed")
- Cooldown detection uses `meta_ad_performance` data age — if no campaign data older than 7 days exists, market is in cooldown

## Decisions made
- **5 concepts for cold start** (not all 9): Strategy engine docs say 3-5 for cold start. 5 gives diversity without spreading too thin.
- **200 SEK/concept** for budget recommendation (above 150 learning floor): gives breathing room
- **Cooldown based on performance data age** (not explicit tracking): avoids new tables/columns. If ALL ad set data for a market is < 7 days old, it's in cooldown. Natural — no state to maintain.
- **Cooldown blocks entire market** (both image + video formats): simpler than per-format cooldown, and makes sense — the market needs time to learn.
