## Decisions made
- **Per-concept learnings** (not aggregated patterns) — each concept-market combo gets its own learning record. Pattern detection happens at query time via the API
- **Auto-save silently** — no approval needed for AI-generated learnings. User never has to write anything
- **Both winners and losers** — learnings generated for all terminal outcomes
- **Dual injection** — learnings feed into brainstorm prompts (AI context) AND show visually on brainstorm page (user context)
- **Claude Haiku 4.5** for learning generation — lightweight, fast, cheap (~$0.001 per learning)
- **Separate table** from existing `ad_learnings` — that's an operational event log, not structured testing knowledge
