## Decisions made
- **Cheerio over regex for HTML attribute manipulation** — Regex can't handle data attributes with JSON values (embedded quotes). Cheerio is already a dependency.
- **Random nonce for placeholders** — Prevents collision if source content contains strings matching `__x0__` pattern. Low probability but zero-cost prevention.
- **"Save failed" error state** — Added `"error"` to `AutoSaveStatus` union type. Shows in status bar so user knows save didn't persist.
