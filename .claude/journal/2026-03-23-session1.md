## Decisions made
- **12h timeout for translation quality gate** — 48h was too conservative. The pipeline-push cron runs daily at 03:00 UTC, so concepts would sit for 2 days. 12h means same-day or next-morning push.
- **500 SEK min spend threshold for ROAS-based page ranking** — prevents low-data pages from dominating due to small sample size
- **Launchpad queue thresholds** (0-2/3-5/6-9/10+) — balances concept freshness with not overwhelming the system. 10+ means plenty of untested creative, no need to generate more.
- **Did NOT implement push timing changes** — user correctly noted night scheduling is common practice (Meta budget resets at midnight). Current 03:00 UTC push time is fine.
- **Did NOT parallelize 9:16 outpainting** — 9:16 uses the translated 4:5 as reference, so true parallelization isn't possible. User caught this.
