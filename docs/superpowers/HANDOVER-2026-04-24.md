## Key design decisions / memory

### Env var gotcha

`ANTHROPIC_API_KEY` was set to EMPTY STRING in the user's shell. Next.js respects shell env over `.env.local`. Dev server must start with `unset ANTHROPIC_API_KEY` before `npm run dev`, OR the key needs to be loaded from `.env.local` explicitly.

The worktree's `.env.local` was initially a symlink to `../../.env.local` which didn't work reliably. Current state: real copy at `.worktrees/quiz-builder-editor/.env.local`. Don't replace with symlink.

### Streaming required for Claude

Anthropic SDK rejects `messages.create()` when max_tokens is high enough to imply >10-min operation. Use `client.messages.stream().finalMessage()` pattern instead. quiz-adapt.ts already does this.

### Cost per adapt

- Adapt call: ~21k input tokens / ~30k output tokens = ~$0.12 input + $0.45 output = $0.57 per adapt at Sonnet 4.5 pricing. Prompt caching cuts input ~90% on subsequent calls for same product.
- Takes ~7 minutes for a 23-step quiz.

### Workspace, market, and CF Pages mapping

Per memory: SE → `halsobladet-blog` CF Pages project → halsobladet.com. DK → `smarthelse`. NO → `helseguiden`. Hydro13 redirects to `get-renew.com/products/hydro13` (Shopify, not in CF Pages).
